# Store Listing Draft

## Title
Nyra Theme Studio

## Short Description
Dark theme presets and simple surface controls for ChatGPT.

## Description
Nyra Theme Studio lets you adjust the visual appearance of ChatGPT with dark presets and simple color controls.

It is built for users who prefer a cleaner, older-style ChatGPT look with calmer dark surfaces. You can choose a preset or adjust the main areas yourself, including the chat background, message bubble, message input box, content panels, and sidebar.

Features:
- Dark theme presets
- Older-style ChatGPT-inspired gray/dark look
- Live color customization
- Separate controls for chat background, message bubble, input box, content panels, and sidebar
- Local-only theme settings
- Lightweight CSS-based styling
- No account connection required
- No data collection

Nyra Theme Studio only changes the visual appearance of supported ChatGPT pages. It does not modify ChatGPT functionality, automate conversations, read your messages for analytics, or send your data to an external server.
